# Условные задания — Post-game

Всё, чего **нет** в этом репозитории, и почему. Каждая строка — сознательно отложенный кусок с известным швом подключения.

Правило: **пока строка здесь — в репозитории не должно быть ни строчки кода, ни хука, ни поля конфига под неё.** «Подготовить место» — это и есть спекулятивная обобщённость, которую запрещает `standards.md`.

Реализовал — удали строку отсюда.

## Никогда

Не «пока», а **несовместимо с построением**. Изменение любого пункта — новая игра, не правка этой.

| Что | Почему |
|---|---|
| **Перенос состояния между версиями канистры** | новая канистра = новый ключ = чужие эскроу. Старая доживает свои задания сама (game-spec §10) |
| **Апелляции, пересмотр вердикта** | один эскроу — один вердикт навсегда. Пересмотр = второй валидный исход, его не существует |
| **Репутация за голосование, награды голосующим** | репутация ≡ честно уплаченный доллар через сплиттер. Другой эмиссии нет и не будет |
| **Кворум, делегирование, квадратичное голосование** | правило зафиксировано: строгое большинство веса, иначе `cancel`. Другая схема = другая игра |
| **Вес голоса поверх кластеров** | книга локальна кластеру (core-spec §10.3). Склейка адресов никогда не питает ни книгу, ни голос |
| **Текст задания в канистре** | текст — модерируемый контент, ему место на сервере. В канистре только хеш |
| **Приз, множитель, ставки** | из эскроу выходит ровно `GROSS`. Заработать больше внесённого нельзя никому |
| **Своя форма эскроу под эту игру** | двух исходов достаточно (factory-spec §7). Новая форма — решение `crown-factory`, не игры |

## Что сломается первым, и почему это нормально

Канистра — единственный участник игры, который может умереть: кончатся циклы, замолчит подсеть, найдётся баг. Ни один из этих сценариев не теряет деньги: каждый эскроу выходит через `refund()` по своему `DEADLINE`, без подписи, транзакцией кого угодно.

Баг в заблэкхоленной канистре неисправим — как и в ядре, митигация не апгрейд, а замена: новая канистра с новым ключом принимает новые задания, старая доживает открытые. Репутация не страдает вовсе: она живёт в книге ядра и питается только сеттлментами, которые уже случились.

Поэтому блэкхол дёшев, а G5 спокойно ждёт готовности всей платформы.
